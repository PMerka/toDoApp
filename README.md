# Base for new stack testing

Fullstack node app for testing tech stacks:
BE, FE Docker and deployment to coolify.

## App initialization:

create .env file in the BE folder

by docker just run:

```
docker compose up
```

or run locally BE and FE

BE:

```
npm install
npx prisma generate
RUN npx prisma migrate deploy

npm run dev

or

node index.js
```
